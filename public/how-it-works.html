How to Use the Interactive Table

Welcome! Here is how you can make the most of this interactive periodic table:

1. Selecting Elements

Simply click or tap on any element card in the main grid to open the detail view. You can also use the search bar to find elements by name, symbol, or atomic number.

2. 3D Visualizations

Each element features a 3D Bohr model and a Crystal Structure model. You can rotate and zoom these models to understand the physical arrangement of atoms and electrons.

3. Filtering

Use the "Filter by Group" dropdown to highlight specific categories of elements, such as Alkali Metals, Noble Gases, or Lanthanides.

4. Deep Linking

You can share a direct link to any element by copying the URL when the element detail is open. For example, ?element=Au will take someone directly to Gold.
