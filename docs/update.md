# Update Notes

This updated version brings significant improvements to the stability and educational experience of the Interactive Periodic Table. We have focused on refining the mobile experience and ensuring the highest scientific accuracy for our users.

## Enhanced Scientific Terminology
- Chemical series terminology has been standardized across the application to provide a more consistent academic experience.
- Updated terminology for Lanthanide and Actinide series to align with modern educational standards.

## Improved Pronunciation Logic
Refined the audio pronunciation engine for improved reliability across all devices.
- Added robust handling for voice pre-loading and asynchronous availability, ensuring that element name pronunciations are crisp and available whenever you need them.
- Added an intelligent delay mechanism between speech cycles to prevent audio cutting on certain mobile environments.

## Element Deep-Dives
- Massive expansion of the element information database.
- Every element now includes comprehensive summaries covering real-world industrial uses, history of discovery, production methods, and detailed health and safety data.

## Mobile Stability and Performance
- Fixed a bug where the layout could shift or flicker during orientation changes on mobile devices.
- Improved the stability of the table view when switching between portrait and landscape modes.
- Optimized zoom behavior logic in the main application to ensure a smooth, precise view of the table at any scale.
- Removed redundant CSS to improve overall rendering speed and reduce initial load times.

## Visual and UI Refinements
- Standardized the global navigation system for a more consistent feel throughout the app.
- Updated the footer and "Back to Home" navigation for more intuitive transitions during deep-research sessions.
